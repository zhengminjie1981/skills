## product-pitch（产品介绍）

### 结构

| 页面 | 内容 | 布局 |
|------|------|------|
| 1 | 封面（产品名 + 标语） | 居中 |
| 2 | 痛点（用户面临的问题） | 列表 |
| 3 | 方案（产品如何解决） | 图文 |
| 4 | 优势（对比竞品） | 三栏 |
| 5 | 演示（截图/视频） | 大图 |
| 6 | CTA（行动号召） | 居中 |

### 适用场景

- 产品发布会
- 客户演示
- 融资路演
- 内部推广

### 自定义提示

- 替换 `{{产品名}}` 为实际产品名
- 痛点部分使用真实用户反馈
- 优势部分准备数据对比

## 深浅主题切换

### 方法1: 直接使用模板

模板文件中已包含 dark 和 light 两个版本：

```markdown
<!-- 使用深色版本 -->
复制 product-pitch-dark 部分

<!-- 使用浅色版本 -->
复制 product-pitch-light 部分
```

### 方法2: 修改 frontmatter

```yaml
<!-- 深色 -->
slideOptions:
  theme: night

<!-- 浅色 -->
slideOptions:
  theme: white
```

### 方法3: 自定义 CSS

参考 `references/themes.md` 中的配色方案。

## 使用模板的流程

```
1. 确定模板类型 → 选择对应模板文件
2. 选择深/浅主题 → 复制对应版本
3. 替换占位符 → {{产品名}}、{{日期}} 等
4. 填充内容 → 按页面说明填写
5. 预览调整 → 检查布局和样式
6. 演示/导出 → 使用 Slides Extended 演示
```
