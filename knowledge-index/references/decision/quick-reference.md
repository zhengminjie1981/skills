## 配置速查

### 最小配置

```yaml
indexing:
  file_types:
    - ".md"
    - ".pdf"
    - ".docx"
```

### 常用配置

```yaml
indexing:
  file_types:
    - ".md"
    - ".pdf"
    - ".docx"

  exclude:
    - "_index.yaml"
    - ".git"
    - "drafts/"

  summary:
    max_length: 300
    include_keywords: true

  update_detection:
    method: "mtime"
```

### 高级配置

```yaml
indexing:
  file_types:
    - ".md"
    - ".pdf"
    - ".docx"

  exclude:
    - "_index.yaml"
    - ".git"
    - "*.tmp"

  summary:
    max_length: 500
    include_keywords: true
    keywords_count: 8

  update_detection:
    method: "hash"

  performance:
    batch_size: 50
    parallel_workers: 4
```

## 工具选择速查

| 场景 | 推荐工具 | 原因 |
|------|---------|------|
| 高质量摘要 | doc2md skill | AI 增强，质量最高 |
| 大规模知识库 | PyMuPDF | 速度最快 |
| 复杂格式 | pandoc | 兼容性最好 |
| 仅 Markdown | 直接读取 | 无需额外工具 |

## 变更检测方法速查

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| mtime | 快速 | 可能误判 | 大型知识库（>100 文档） |
| hash | 精确 | 较慢 | 小型知识库（<100 文档） |
| 混合 | 平衡 | 复杂 | 中型知识库 |

## 摘要质量速查

| 文档类型 | 推荐长度 | 重点 |
|---------|---------|------|
| 技术文档 | 200-300 字符 | 代码示例、配置步骤 |
| 业务文档 | 150-200 字符 | 关键指标、业务流程 |
| 学术文档 | 300-500 字符 | 研究方法、主要结论 |
| 手册指南 | 200-250 字符 | 使用步骤、注意事项 |

## 错误处理速查

| 错误 | 快速解决方案 |
|------|-------------|
| 路径不存在 | 检查路径拼写，使用绝对路径 |
| 工具不可用 | 安装 doc2md/pandoc/PyMuPDF |
| 转换失败 | 跳过该文档，继续处理 |
| 索引损坏 | 重建索引 |
| 检索无结果 | 更新索引，优化关键词 |

## 性能优化速查

| 问题 | 解决方案 |
|------|---------|
| 构建慢 | 增加批处理大小，使用快速工具 |
| 内存高 | 减小批处理大小，禁用缓存 |
| 检索慢 | 优化关键词，使用主题聚类 |
