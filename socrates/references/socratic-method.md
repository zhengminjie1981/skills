# 苏格拉底式提问方法

## 六种问题类型

### 1. 澄清概念

**模板**:
- 「你说的是 X 还是 Y？」
- 「能举个具体例子吗？」
- 「这个概念对你意味着什么？」

### 2. 探测假设

**模板**:
- 「这背后的假设是什么？」
- 「如果这个假设不成立，结论会怎样？」
- 「我们是否在把「习惯」当成「必然」？」

### 3. 追问证据

**模板**:
- 「有什么支持这个观点？」
- 「有没有相反的例子？」
- 「你有多大把握？」

### 4. 视角切换

**模板**:
- 「反对者会怎么说？」
- 「局外人怎么看？」
- 「一年后回看，你会怎么评价？」

### 5. 后果推演

**模板**:
- 「如果是这样，会导致什么？」
- 「最坏情况是什么？」
- 「这个决定会关闭哪些可能性？」

### 6. 质疑问题

**模板**:
- 「我们真正想解决的是什么？」
- 「这是表面问题还是根本问题？」

## 常见陷阱

| 陷阱 | 错误示范 | 正确做法 |
|------|----------|----------|
| 诱导性提问 | 「你不觉得 X 更好吗？」 | 「X 和 Y 各有什么优缺点？」 |
| 连续追问 | 一口气问 3+ 个问题 | 每次只问一个 |
| 预设答案 | 「是因为 A 吗？」 | 「原因可能是什么？」 |
| 过早评判 | 「这个想法有问题」 | 「让我们看看这个假设是否成立」 |
| 好为人师 | 「其实你应该...」 | 「你考虑过...吗？」 |

## 提问节奏

```
❌ 「你的目标是什么？为什么选这个？有没有考虑过其他？」
✅ 「你的目标是什么？」（等待回答）
```

```
用户：「...我觉得可能是 A」
→ 「嗯...你说「可能是」，是什么让你不确定？」（跟进而非跳跃）
```
