# 数据结构规范

> **文档定位**：本文档属于 L2 规范级，回答"跨模块数据结构的权威定义"。
> - 上位文档：[系统架构设计](../01-系统设计/系统架构设计.md)
> - 下游文档：各模块设计文档

## 1. 核心数据结构

### 1.1 <数据结构名称>

**定义**：
```python
from dataclasses import dataclass
from typing import List, Optional

@dataclass
class FeatureExtractionInput:
    """特征提取模块输入数据结构"""
    image_data: np.ndarray      # 图像数据，形状 (H, W, C)
    metadata: dict              # 元数据
    config: Optional[dict]      # 可选配置
```

**字段说明**：
| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| image_data | np.ndarray | 是 | 图像数据，形状 (H, W, C)，dtype=uint8 |
| metadata | dict | 是 | 元数据，必须包含 "timestamp" 字段 |
| config | dict | 否 | 配置参数，默认为 None |

**约束条件**：
- image_data 的形状必须是 (H, W, 3)
- metadata["timestamp"] 必须是 ISO 8601 格式

**使用示例**：
```python
input_data = FeatureExtractionInput(
    image_data=image,
    metadata={"timestamp": "2026-03-03T10:00:00Z"},
    config={"algorithm": "SIFT"}
)
```

- [特征提取模块](../03-模块设计/特征提取模块.md)
- [预处理模块](../03-模块设计/预处理模块.md)

```markdown
# 接口规范

> **文档定位**：本文档属于 L2 规范级，回答"跨模块接口的权威定义"。

## 1. 模块间接口

### 1.1 <接口名称>

**接口签名**：
```python
def extract_features(
    input_data: FeatureExtractionInput,
    algorithm: str = "SIFT"
) -> FeatureExtractionResult:
    """
    特征提取接口

    Args:
        input_data: 特征提取输入数据
        algorithm: 特征提取算法，可选 "SIFT", "ORB", "SURF"

    Returns:
        FeatureExtractionResult: 特征提取结果

    Raises:
        ValueError: 输入数据格式不正确
        RuntimeError: 特征提取失败
    """
```

**参数说明**：
| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| input_data | FeatureExtractionInput | 是 | - | 输入数据 |
| algorithm | str | 否 | "SIFT" | 算法选择 |

**返回值**：
- 类型：FeatureExtractionResult
- 说明：见 [数据结构规范](数据结构规范.md#featureextractionresult)

**异常**：
| 异常类型 | 触发条件 | 处理建议 |
|---------|---------|---------|
| ValueError | 输入数据格式不正确 | 检查输入数据 |
| RuntimeError | 特征提取失败 | 查看日志，重试 |

**使用示例**：
```python
from features import extract_features
from data_structures import FeatureExtractionInput

input_data = FeatureExtractionInput(...)
result = extract_features(input_data, algorithm="ORB")
```
