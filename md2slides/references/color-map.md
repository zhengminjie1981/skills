## 透明度

语法：`rgba(R, G, B, A)`，A 为 0～1

| 说法 | CSS 值 |
|------|------|
| 半透明白色 | `rgba(255,255,255,0.5)` |
| 微透明黑色 | `rgba(0,0,0,0.15)` |
| 半透明深蓝 | `rgba(26,58,107,0.6)` |
