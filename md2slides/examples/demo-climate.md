# 2024：史无前例的气候红线

- ⚠️ 2024年是 **175年有记录以来最热的一年**
- 📈 全球平均气温较工业化前水平高出 **1.55°C**，首次突破1.5°C阈值
- 🌏 阿拉伯地区升温速度是全球平均的 **2倍**
- 📊 极端天气事件频率与强度双双创历史新高
- ⚙️ 北极海冰面积持续萎缩，2024年夏季创新低纪录
- 🔒 若气温升幅超过 **2°C**，地球部分生态系统将面临不可逆转的崩溃

# 主要国家碳中和承诺

- 🌏 **中国**：2030年前碳达峰，2060年前实现碳中和；2035年较峰值减排7-10%
- ⚙️ **欧盟**：2030年减排 **55%**（对比1990年），2050年碳中和
- ⚠️ **美国**：特朗普政府撤销2030/2035 NDC目标，重回高碳轨道
- 🚀 **印度**：2070年碳中和，可再生能源占比目标 **50%**（2030年）
- 💡 **日本/韩国**：2050年碳中和，绿氢产业列为优先战略
- 📊 全球已有 **140余个国家**提出碳中和或净零目标

# 能源转型：唯一可行路径

- ⚡ 2024年全球可再生能源装机容量突破 **3,870 GW**
- 📈 太阳能成本10年间下降 **90%**，成全球最便宜的发电方式
- 🏆 全球电动汽车保有量突破 **4,000万辆**，每年仍以40%速度增长
- 💰 2024年全球清洁能源投资达 **1.7万亿美元**，首次超过化石能源
- ⚙️ 绿氢成本持续下降，2030年有望实现平价
- 🔒 煤电退出速度仍不足，是实现1.5°C目标的最大障碍
