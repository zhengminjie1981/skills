# 2024年：创纪录的一年

- 📈 全年新能源汽车销量 **1286.6万辆**，同比增长 **35.5%**
- 🏆 NEV渗透率突破 **40.9%**，每10辆新车超4辆是新能源
- 🌏 中国占全球新能源汽车销售的 **70%**，绝对主导地位
- 💰 仅12月单月销量即达 **159.6万辆**，创单月历史纪录
- 🚀 中国自主品牌合计销量 **1807.9万辆**，市占率 **65%**
- 📊 外资品牌市场份额持续下滑，合资车企压力显著加剧

# 头部车企核心竞争力

- ⚙️ **比亚迪**：刀片电池 + 垂直整合，全产业链自研
- 🚀 **华为系（问界/智界）**：智能驾驶领先，鸿蒙车机生态
- 💡 **理想汽车**：增程+纯电双路线，家庭用户精准定位
- 📈 **小米SU7**：2024年交付超13万辆，跨界造车首战告捷
- 🌏 **吉利/极氪**：高端化突破，出口欧洲市场持续发力
- 🤝 **蔚来**：换电模式独树一帜，BaaS订阅模式创新

# 全球竞争：中国制造走向世界

- 🌏 中国NEV出口 **177.3万辆**，同比增长 **12.8%**
- 📈 比亚迪已在 **70余个国家和地区**销售
- ⚠️ 欧盟对中国电动车加征关税，最高达 **45.3%**
- 🏆 中国动力电池占据全球 **60%以上**的市场份额
- 💰 宁德时代全球市占率稳居第一，达 **37.9%**
- 🚀 "出海"成为2025年各大车企最重要战略方向
