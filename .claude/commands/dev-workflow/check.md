# 验证文档一致性

检查项目文档之间的一致性，确保引用、定义和描述保持同步。

## 工作流

1. **扫描文档** - 扫描项目文档目录，识别所有技术文档
2. **检查引用** - 验证文档间的引用是否有效
3. **验证一致性** - 检查：
   - 数据结构定义是否一致
   - 接口规范是否一致
   - 模块边界描述是否一致
   - 版本历史是否完整
4. **生成报告** - 输出一致性检查结果和改进建议

## 检查项目

- [ ] 系统架构与模块设计一致性
- [ ] 数据结构规范与代码一致性
- [ ] 接口规范与实现一致性
- [ ] 文档引用有效性
- [ ] 版本历史完整性
