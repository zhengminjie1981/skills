# 已有 Skill 符合性检查清单

> 用于检查现有 skill 是否符合规范，并指导优化。

## 1. 目录结构检查

- [ ] 有 `SKILL.md` 入口文件
- [ ] `references/` 仅包含 `.md` 文档和配置模板
- [ ] 可执行代码在 `scripts/` 目录
- [ ] 开发文档在 `docs/` 目录（如有）
- [ ] 示例文件在 `examples/` 目录（如有）
- [ ] 测试文件在 `tests/` 目录（如有）
- [ ] 运行时数据在 `data/` 目录（如有）

**检查命令**：
```bash
# 检查 references/ 中是否有可执行代码
find skill-name/references -name "*.py" -o -name "*.sh"
# 应该无输出
```

## 2. 代码位置检查

- [ ] `*.md` 文件中没有实际执行代码
- [ ] 代码块仅作为示例（展示用法）
- [ ] Python 脚本在 `scripts/*.py`
- [ ] Shell 脚本在 `scripts/*.sh`

**检查要点**：
```markdown
<!-- ✅ 允许：示例代码 -->
\`\`\`bash
python scripts/tool.py --option
\`\`\`

<!-- ❌ 不允许：实际执行代码直接写在 md 中 -->
\`\`\`python
def main():
    # 实际逻辑...
\`\`\`
```

## 3. Front Matter 检查

- [ ] 有 `name` 字段（小写字母和连字符）
- [ ] 有 `description` 字段
- [ ] `description` 包含触发场景
- [ ] `description` 包含触发关键词
- [ ] `description` 包含不触发场景（如有必要）
- [ ] 没有多余的 YAML 字段

**示例**：
```yaml
---
name: skill-name
description: |
  简短描述。

  **触发场景**（包含以下关键词）：
  - 场景1：关键词A、关键词B

  **不触发场景**：
  - 场景X：关键词Y
---
```

## 4. 按需加载检查

- [ ] `SKILL.md` < 500 行
- [ ] 详细内容在 `references/` 中
- [ ] 有参考资源索引（任务→文档映射）
- [ ] 开发文档不在 `references/` 中

**检查命令**：
```bash
# 检查 SKILL.md 行数
wc -l skill-name/SKILL.md
```

## 5. AI 友好性检查

- [ ] 有核心功能表格
- [ ] 有决策树或流程图
- [ ] 有默认值说明
- [ ] 有任务→文档映射表
- [ ] 标记了特殊情况

**AI 友好元素**：
```markdown
<!-- 功能表格 -->
| 功能 | 说明 | 触发示例 |
|------|------|---------|

<!-- 决策树 -->
问题？
├─ YES → 路径A
└─ NO  → 路径B

<!-- 默认值 -->
### Safe Defaults
- `--option`（推荐）

<!-- 任务映射 -->
| 任务 | 读取文档 |
|------|---------|

<!-- 特殊情况 -->
### Red Flags
- 场景X → 使用Y
```

## 6. 文档质量检查

- [ ] 有一行总结
- [ ] 有快速开始（可执行命令）
- [ ] 有典型场景示例
- [ ] 有常见问题（如有）
- [ ] 使用 UTF-8 编码
- [ ] 使用 LF 换行符

## 优化优先级

### 必须修复

| 问题 | 影响 | 操作 |
|------|------|------|
| Front matter 缺少触发场景 | 无法正确触发 | 补充 description |
| 可执行代码在 md 中 | 违反规范 | 移动到 scripts/ |

### 建议修复

| 问题 | 影响 | 操作 |
|------|------|------|
| SKILL.md > 500 行 | 加载过重 | 拆分到 references/ |
| 开发文档在 references/ | 占用 token | 移动到 docs/ |
| 缺少不触发场景 | 可能误触发 | 补充边界说明 |

### 可选优化

| 问题 | 影响 | 操作 |
|------|------|------|
| 缺少决策树 | AI 判断困难 | 添加决策树 |
| 缺少默认值 | 选择困难 | 添加默认值说明 |
| 缺少任务映射 | 定位困难 | 添加映射表 |

*完成检查后，按优先级依次优化*
